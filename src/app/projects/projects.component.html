Daniele ti fa il CV

A windows only (for now) application to create and manage a curriculum vitae using Electron. The app works by starting a local backend server (written in ASP.NET) on startup which manages an API to generate the PDF file for the CV and to manipulate user data. The data is displayed and can be modified by the user via the UI (written in Angular)

Angular Game of Life Background

An angular 19+ component that plays Conway's Game of Life while sitting in the background of the page
